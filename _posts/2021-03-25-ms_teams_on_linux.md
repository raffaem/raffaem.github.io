I finally found a way to have Microsoft Teams run on Linux using free software.

Use [Brave](https://brave.com/) browser and allow third-party cookies (otherwise Miscrosoft Teams will not work).

Teams also works on proprietary Google Chrome (but _not_ on Chromium), 
Microsoft Edge (which now also comes in [Linux flavor](https://blogs.windows.com/msedgedev/2020/10/20/microsoft-edge-dev-linux/))
or on Microsoft proprietary Linux desktop client, which is available also on [Flathub](https://flathub.org/apps/details/com.microsoft.Teams)
(there is also an unofficial desktop client which is basically 
a wrapper around the website [here](https://flathub.org/apps/details/com.github.IsmaelMartinez.teams_for_linux),
although I haven't tested it).
